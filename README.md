# React + Vite Проект

Этот проект создан на основе **React** и **Vite** для быстрой и эффективной разработки современных веб-приложений.

## 📋 Описание структуры проекта

- `index.html` — главный HTML-файл, точка входа в приложение.
- `package.json` — файл конфигурации проекта: зависимости, скрипты запуска и сборки.
- `vite.config.js` — настройки сборщика Vite (плагины, порт, алиасы и т.д.).
- `.gitignore` — список файлов и папок, которые не будут попадать в репозиторий Git.
- `src/` — исходный код приложения:
  - `main.jsx` — точка входа для рендеринга React-приложения.
  - `App.jsx` — главный компонент приложения.
  - `App.css` — стили компонента `App`.
  - `index.css` — глобальные стили проекта.

## 🛠 Что нужно установить локально

Перед началом работы убедитесь, что у вас установлены следующие инструменты:

### 1. **Node.js** (включает npm)
- Скачайте и установите с официального сайта: [https://nodejs.org/](https://nodejs.org/)
- Рекомендуется версия **LTS** (Long Term Support).
- Проверка установки:
  ```bash
  node -v
  npm -v
  ```

### 2. **Редактор кода**
- Рекомендуется **IntelliJ IDEA Ultimate** (с плагинами для React, Prettier, ESLint).
- Альтернативы: VS Code, WebStorm.

### 3. **Git** (опционально, если планируете работать с репозиторием)
- Скачать: [https://git-scm.com/](https://git-scm.com/)
- Проверка:
  ```bash
  git --version
  ```

## 🚀 Быстрый старт

### 1. Установка зависимостей
После клонирования или создания проекта выполните в терминале:
```bash
npm install
```
Эта команда установит все необходимые пакеты из `package.json`.

### 2. Запуск сервера разработки
```bash
npm run dev
```
После запуска откройте в браузере: `http://localhost:5173/`

> При изменении файлов проект будет автоматически обновляться в браузере.

### 3. Сборка проекта для продакшена
```bash
npm run build
```
Готовые файлы появятся в папке `dist/`.

### 4. Предпросмотр собранной версии
```bash
npm run preview
```

## 💡 Советы для новичков

1. Начните с изучения основ **JavaScript**.
2. Освойте базовые концепции **React**: компоненты, props, state, хуки (`useState`, `useEffect`).
3. Практикуйтесь в написании **CSS** (Flexbox, Grid).
4. Используйте `console.log()` для отладки.
5. Не бойтесь экспериментировать — изменения в коде сразу видны в браузере!
